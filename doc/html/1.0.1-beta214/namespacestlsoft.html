stlsoft Namespace Reference

Detailed Description

The STLSoft namespace - stlsoft.

The Pantheios project inserts overloads of the c_str_data_a and c_str_len_a String Access Shims into the stlsoft namespace, for generalised manipulation of various Pantheios types, including pantheios::pan_slice_t, pantheios::integer, pantheios::pointer, and pantheios::real.

Because Shims are unbound sets of functions, opening and extending the stlsoft namespace in this way is quite legitimate.
